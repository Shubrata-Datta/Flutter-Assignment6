# 📱 Flutter Assignment 6

## 📌 Project Overview
This repository contains a Flutter-based assignment project developed to practice modular programming and Dart file organization. The project demonstrates how to structure a Flutter application by separating logic into multiple Dart files and integrating them through the main application.

---

## 🎯 Objectives
- Create and manage a Flutter project  
- Organize code into multiple Dart files  
- Implement and test separate tasks in a structured way  
- Practice importing and using custom Dart files in `main.dart`  
- Upload and manage the project using GitHub  

---

## 🗂️ Project Structure
lib/
│── main.dart # Entry point of the application
│── a1_task1.dart # Contains Task 1 implementation
│── a1_task2.dart # Contains Task 2 implementation

---

## ⚙️ Features
- Modular code organization using separate Dart files  
- Easy testing by calling functions in `main.dart`  
- Clean and maintainable project structure  
- Beginner-friendly Flutter setup  

---

## 🚀 How It Works
- Task-specific logic is written in:
  - `a1_task1.dart`
  - `a1_task2.dart`
- These files are imported into `main.dart`
- Functions are executed from `main.dart` to display/test output
